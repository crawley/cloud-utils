QCIF Cloud utilities
====================

Utilities for working with QCIF cloud VM instances.

**These scripts are under construction!**

Scripts
-------

### q-gui-setup.sh

Setup X-Windows plus a VNC server.

### q-storage-setup.sh

Setup a NFS client to mount RDSI storage.
